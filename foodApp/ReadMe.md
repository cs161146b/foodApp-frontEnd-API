# Food Recipe App (React)

## Overview
This is a frontend web application built with React that allows users to search for food recipes using an external API and view detailed recipe information.

The project demonstrates React fundamentals, API integration, and dynamic search functionality.


## Features

- Search recipes by keyword using external API
- View recipe details (ingredients & instructions)
- Dynamic rendering of search results
- Responsive UI built with React components

---

## Tech Stack

- React
- JavaScript (ES6+)
- HTML5 / CSS3
- REST API integration


## API Usage

The application fetches recipe data from an external recipe API based on user search input.

 Note: API may have request limits depending on usage.
